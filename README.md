
# Calendar
A simple calendar that can be used from the CLI.

## Dependencies
1. rspec
Rspec was used here for testing
```
gem install rspec
```
2. colorize
```
gem install colorize
```

## Versions
### Ruby version
```
ruby 1.7.1
```
